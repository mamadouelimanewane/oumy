# 🇸🇳 SenFood - Plateforme de Livraison de Repas (Sénégal)

## Présentation
SenFood est une application de livraison de repas style UberEats dédiée au marché Sénégalais. Ce projet permet aux clients de commander des repas, aux restaurants (y compris les Dark Kitchens) de gérer leurs ventes, et aux livreurs d'optimiser leurs courses.

## Les 4 Modules du Projet
1. **Client** PWA (React.js)
2. **Dashboard Restaurant** Web (React.js)
3. **App Livreur** Mobile (Android, React Native)
4. **Panel Admin** Web (Node.js/React.js)

## Stack Technique
*   **Frontend Web** : React.js, TailwindCSS
*   **Frontend Mobile** : React Native (Expo)
*   **Backend** : Node.js (Express)
*   **Base de Données** : PostgreSQL
*   **Temps Réel & Push** : Firebase
*   **Paiement** : Wave & Orange Money

## Intégrations Principales
- Suivi GPS en Temps Réel
- Notifications Push (Clients, Restaurants, Livreurs)
- Passerelles de paiement (Wave API, Orange Money Sandbox/Prod)
